# MarkItDown Batch Converter

Drop-and-run script to convert all PDF and Excel files in a folder
to Markdown, using Microsoft's MarkItDown library.

## Requirements

Python 3.10 or higher.

## Installation

pip install -r requirements.txt

## Usage

1. Drop convert_all.py into the folder with your files
2. Run:

py convert_all.py

3. Find your Markdown files in the markdown_output subfolder.

## Supported formats

- PDF (.pdf)
- Excel (.xlsx, .xls)
